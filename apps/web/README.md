# web (`apps/web`)

Webapp React (Vite + TypeScript + Tailwind + shadcn/ui + React Router).

## Comandi

```bash
pnpm dev
pnpm build
pnpm preview
pnpm lint
pnpm ts:check
pnpm test
```

Nessuna credenziale AWS richiesta per `dev` / `build` locali. Il deploy su S3/CloudFront è fuori scope per US-EP00-05; vedi README root.
